## Summary
 

### Sets
If $\alpha$ is some type ($\alpha : Type*$), then the type of a set $s$ with elements of type \alpha is 
```lean
s : Set α
```

The following table summarizes what the types of the proof terms look like for different goals involving sets. 
To construct proof terms of the correct type, we can use tactics introduced previously.

<table><thead>
  <tr>
    <th>Goal</th>
    <th>Type of proof term</th>
  </tr></thead>
<tbody>
  <tr>
    <td>s ⊆ t</td>
    <td>∀ x : α, x ∈ s → x ∈ t</td>
  </tr>
  <tr>
    <td>x ∈ s ∩ t</td>
    <td>x ∈ s ∧ x ∈ t</td>
  </tr>
  <tr>
    <td>x ∈ s ∪ t</td>
    <td>x ∈ s ∨ x ∈ t</td>
  </tr>
  <tr>
    <td>x ∉ s</td>
    <td>x ∈ s → False</td>
  </tr>
</tbody>
</table>
